This is a Power BI dashboard I created as part of my student portfolio to demonstrate my skills in data analysis and visualization. I built the dashboard using Power Query for data preparation, DAX for calculated fields, and a variety of interactive visualizations.

This project helped me strengthen my Power BI skills and learn how to design clear, stakeholder-friendly dashboards. It reflects my interest in data careers and my ability to turn raw survey data into meaningful insights.
